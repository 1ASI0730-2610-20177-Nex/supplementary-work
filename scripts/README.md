# Online Editors

Los siguientes editores en línea son recomendados para realizar las prácticas del curso.

## CodePen

https://codepen.io

Permite escribir HTML y CSS con vista previa en tiempo real.

---

## JSFiddle

https://jsfiddle.net

Editor en línea para probar código HTML y CSS rápidamente.

---

## Replit

https://replit.com

Entorno de desarrollo en línea con soporte para múltiples lenguajes y proyectos web.
