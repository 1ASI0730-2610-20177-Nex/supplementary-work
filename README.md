# 🌐 Fundamentos del Desarrollo Web para Principiantes

Curso introductorio de desarrollo web dirigido a estudiantes de secundaria (12 a 17 años) sin experiencia previa en programación.

En este curso aprenderás los conceptos básicos de HTML y CSS para crear tu primera página web desde cero, utilizando únicamente herramientas gratuitas y en línea.

Este repositorio contiene el código fuente de las prácticas y el plan completo del curso.

## ¿Qué necesito para empezar?

Solo necesitas:

- Un navegador web (Chrome, Firefox, Safari o Edge).
- Conexión a internet.
- Un editor en línea gratuito.

Editores recomendados:

- CodePen: https://codepen.io
- JSFiddle: https://jsfiddle.net
- Replit: https://replit.com

**No es necesario instalar ningún programa ni crear una cuenta.**

---

## ¿Cómo uso este repositorio?

1. Mira el video de la lección correspondiente.

2. Abre el editor en línea indicado en la descripción del video.

3. Copia el contenido del archivo ubicado en:

```text
starter-files/
```

4. Completa los ejercicios.

5. Si tienes dificultades, revisa la solución en:

```text
completed-examples/
```

## ¿Cómo ejecuto el código?

1. Ingresa a CodePen:

https://codepen.io

4. Los cambios aparecerán automáticamente en la vista previa.
